You've just created a very simple Scala application!  Now lets explore the code, see it run, and makes some changes.

Explore the Code

This very simple application contains just one source file.  To navigate to that file, select "Code" on the left, then select the "app" directory, and select the Hello.scala file.

Now you are seeing source of the "Hello" object.  You'll notice that this object extends the "App" trait which provides a simple way to create an executable application.  The body of the object simply prints "hello, world" to the console.

Run the App

To see the output select Run on the left. The program is automatically run after every build, but if you'd like to re-run, then click the "Start" button. In the logs you should see the "hello, world" message.

Make Changes

To make changes to the application go back to Hello.scala and change "hello, world" to "hello, Builder" and select "Save".

Navigate back to Run and click "Start". You should now see "hello, Builder" displayed.

Test the App

Select Test and this should take you to the test screen, which includes test results. These tests are updated after every build, but you can force them to run by clicking the "start" button.

The application has one test that is currently passing.  The HelloSpec.scala file contains the test code.

Add a new test by adding the following after line 8: 

  "1 + 1" should "equal 2" in {
    (1 + 1) should equal (2)
  }
Save the file and then re-run the tests. You should now have two tests passing.

Next Steps

You're now ready to start your first real application in Scala!

You may continue using the snap ui, or swap to to the snap command line via the snap script in the project directory.
